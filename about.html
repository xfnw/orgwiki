about

after reading some of 100 rabbits' work, i realized that i depend on being always-online too much.

this was created to potentially replace my old pmwiki wiki. pmwiki does not really have any good solutions for working offline, the closest thing being wiki on a stick, which literally runs a webserver with a LAMP stack from a non-portable1 exe file. yuck.

i was also considering using vimwiki, which has a more wiki-like syntax, however that also makes the syntax less consistent and harder to convert to/from. also configuring vimwiki is somewhat a nightmare (perhaps ive been spoiled by lisp's customizability) and it seems a bit overbearing about how it manages files.

1. setupfile

the setupfile is largely based on yeti's brain's setupfile. i know you will probably never read this, but thank you for letting me use it :3

2. contributing

the git repository for this wiki is available in several different places.

send a patch to xfnw ate riseup doot net (replacing ate and doot with the at and the full stop symbol respectively). hopefully i'll get a mailinglist setup Soon™.

if you do not want to send a patch, you can instead open a pull request on github.
