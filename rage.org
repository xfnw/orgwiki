* broken password prompt
rage's fancy pinentry stuff can be a bit buggy, you can make it fall
back to a simpler password prompt by setting the ~PINENTRY_PROGRAM~
env var to nothing

#+begin_example
PINENTRY_PROGRAM= rage -p
#+end_example
